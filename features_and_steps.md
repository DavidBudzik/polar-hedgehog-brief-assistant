# Polar Interactive Brief - Features & Steps Overview

The Polar Hedgehog Brief Assistant is an interactive application designed to guide users through the creation of a comprehensive brand brief. The flow is broken down into thematic sections, each containing multiple guided steps with integrated AI assistance (via Gemini).

Below is a detailed breakdown of the sections, the steps within them, and the specific features available in each step.

---

## 1. Onboarding

### Setup & Kickoff
- **Setup:** Captures the foundational information (Company Name, Project Type, Project Date).
- **Kickoff:** A confirmation bridging the setup with the actual brief questions, establishing the tone for the brief.

---

## 2. Discovery
This section focuses on the structural business information: what the company does, who it competes with, and its unique advantages.

### Problem Statement
- **Features:** 
  - **AI Website Scan:** The user can input their website URL, and the AI will scan it to extract and formulate the core problem the company solves.
  - **Document Upload:** If they don't have a website, users can upload documents (PDF, PPTX, Word) for the AI to analyze and extract the problem.
  - **AI Refinement:** Generates a 1-2 sentence draft that the user can rate (thumbs up/down) or generate 3 new alternatives for.

### Solution Description
- **Features:**
  - **Context-Aware Generation:** AI automatically generates a solution describing how the company solves the previously defined "Problem Statement".
  - **Rating & Alternatives:** Similar to Problem Statement, users can rate the generated solution or request alternative phrasing.

### Competitors
- **Features:**
  - **Manual Entry with AI Tagline:** Users enter competitor names, and the AI fetches or generates a one-line tagline for them.
  - **Categorization:** Competitors can be tagged as "Similar" (e.g., Same ICP) or "Different" (e.g., Niche focus, Better UX). 

### Unique Value Proposition (UVP)
- **Features:**
  - **AI Generation:** Analyzes the problem statement and competitors to automatically generate 4 bold UVP statements.
  - **Rating System:** Users can rate each UVP (🔥 Love it, 😐 OK, ❌ Remove) to filter down to the strongest core propositions.

### Features (Product/Service)
- **Features:**
  - **Generative Suggestions:** AI scans the problem space to suggest 3 key product features (Title & Description).
  - **Manual Definition:** Users can edit AI suggestions or add their own features manually.

---

## 3. Brand Audit
This section reviews existing brand elements and their underlying rationales.

### Company Name Meaning
- **Features:** Text entry explaining the origin or meaning of the company's name.

### Logo Rationale
- **Features:** Captures the story behind the current logo. Includes chips/tags to highlight specific rationale attributes.

### Visual Language Rationale
- **Features:** 
  - Text summary of the visual language.
  - Adjustable sliders rating the brand on scales like Modern, Trustworthy, and Bold.

---

## 4. Brand Direction
This section starts aligning the brand's messaging and core values for the future.

### Keywords
- **Features:** Users select overarching keywords that define the brand's intended tone and personality.

### Brand Messages
- **Features:** Evaluates specific brand messages, allowing the user to approve or reject them based on the selected keywords.

### Core Values (Value Picker)
- **Features:** Selection of primary core values that guide the brand.

### Visual Direction 1 & 2
- **Features:** Two distinct forms to detail two varying creative directions based on earlier selected values. It establishes attributes like Shape (e.g., Organic vs Geometric), Color (Vibrant vs Muted), Motion, and Style.

---

## 5. Visual Preferences
This section captures tangible visual inspirations and hard boundaries.

### Reference Brands
- **Features:** Users enter brands they admire or dislike. They can specify exactly what they "like" or "dislike" about each reference brand to guide the design team.

### Logo Style
- **Features:** Visual selection of different logo styles (Wordmark, Pictorial, Abstract, etc.) to set preferences. Includes an option to remain "open to recommendations".

### Color Palette
- **Features:** 
  - Select an overarching approach to colors.
  - Rate specific color swatches (Like/Dislike) to give the designers defined boundaries.

---

## 6. Summary & Submission
### Summary Review
- **Features:** 
  - A definitive overview displaying all the selected data, keywords, messages, competitors, and references.
  - Final submission trigger, which presents a "Brief Submitted!" success screen with a statistical breakdown of the brief components.
